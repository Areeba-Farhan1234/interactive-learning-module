# Interactive Learning Module

Live Demo: [Interactive Learning Module](https://areeba-farhan1234.github.io/interactive-learning-module/)

## Project Description

An interactive learning module website designed to deliver educational content in an engaging way. It includes lessons, quizzes, multimedia content, and interactive elements to enhance the learning experience.

## Features

- Responsive design for desktop and mobile  
- Multiple lesson pages with text, images, and videos  
- Interactive quizzes for self-assessment  
- Navigation menu for easy lesson access  
- Clean UI and user-friendly layout
